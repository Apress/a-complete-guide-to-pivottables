Book Title

A Complete Guide to PivotTables: A Visual Approach

Issue

I can't download the file Visitors.xls to make exercises as written on the book.

Solution

1. Go the Apress Web site at http://www.apress.com.

2. Point to Quick Links, and then click Source Code/Download.

3. In the Please Select a Title list, click “A Complete Guide to PivotTables: A Visual Approach," and then click Submit.

4. Click Download Source Code File.

5. Click to Open or Save the .zip file.

6. Inside of the .zip file, open the SampleFiles folder, and then open the Visitors.xls file in Excel."
